## Code Quiz Challenge

A timed quiz that will test the coding knowledge of users.  As well as fulfill the following criterias:

* When the "start" button is click the timer will start and the question will be shown.
* After answering the question, the next question will be shown.
* If a question is answered incorrectly,ten seconds will be deducted from the timer.
* If all questions ae answered or the timer reaches 0, than the game will be over.
* After the game is over the user can save their initials and high score.

## Built With
* HTML
* CSS
* JavaScript
